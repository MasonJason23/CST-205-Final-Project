About This Site


This program will take photographs of animals in a non-envasive way via Raspberry Pi, Flask, and OpenCV.
In addition, the project will display images taken via user's local website. It displays an interactive
search bar that allows user's of this project to search for specific animals that the Raspberry Pi has
taken or other existing images.


Learn more about our project by looking at our GitHub !
